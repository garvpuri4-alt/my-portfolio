# Garv Puri — Portfolio

A modern, finance-focused personal portfolio for **Garv Puri** — Finance student at Kennesaw State University, aspiring Investment Banking & FP&A Analyst.

**Live Site:** _coming soon via Vercel_

---

## Tech Stack
- React 19 + React Router
- Tailwind CSS + shadcn/ui components
- Lucide React icons
- Playfair Display + Inter + JetBrains Mono typography

## Run Locally
```bash
yarn install
yarn start
```
App runs at `http://localhost:3000`.

## Build
```bash
yarn build
```
Production-ready static files will be generated in `/build`.

## Deployment (Vercel)
1. Push this folder to a GitHub repo
2. Import the repo on [vercel.com](https://vercel.com)
3. Framework preset: **Create React App** (auto-detected)
4. Build command: `yarn build`
5. Output directory: `build`
6. Click **Deploy**
